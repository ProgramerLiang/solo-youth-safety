# 独行青年安全守护应用（项目草案）

## 项目简介
本项目面向独行青年的人身安全场景，目标是提供及时报警、追踪定位、检测偷拍与安全导航等能力。

## 当前阶段
- 需求梳理中（见 `神秘组织内部资料.md`）
- 已初始化 Git 仓库与 Pi 项目配置
- 已完成 MVP 技术骨架：`frontend`（React + Vite） + `backend`（FastAPI）

## MVP 目标（建议）
1. 一键 SOS 报警
2. 实时定位与轨迹回放
3. 紧急联系人通知

## 下一步开发建议
- 确定技术栈（Web / Mobile / Backend）
- 初始化代码工程
- 定义接口与数据模型
